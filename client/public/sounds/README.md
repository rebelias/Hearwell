# Background Sound Files

Place your background sound files here:

- `cicadas.mp3` - Cicada sounds
- `birds.mp3` - Bird sounds
- `ocean.mp3` - Ocean wave sounds
- `rain.mp3` - Rain sounds

**Requirements:**

- Format: MP3 (preferred) or WAV
- Duration: 10-30 seconds (will be looped)
- Sample rate: 44.1kHz or 48kHz
- Bitrate: 128kbps or higher recommended
- License: Must be CC0/public domain or you have rights to use

**Sources for free sounds:**

- freesound.org (filter by CC0 license)
- pixabay.com (free for commercial use)
- zapsplat.com (free with attribution)

The application will automatically fall back to procedural generation if files are not found.
